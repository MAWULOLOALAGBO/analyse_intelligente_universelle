# 🧠 Data Brain Assistant

Assistant universel d’analyse de données : tout fichier, toute question.

## Fonctionnalités

- Chargement intelligent de fichiers (CSV, Excel, JSON, Parquet, TXT)
- Inférence douce des types
- Compréhension des requêtes en langage naturel
- Validation stricte des opérations
- Exécution modulaire (statistiques, visualisations, nettoyage, exploration)
- Interface Streamlit moderne et intuitive
